Projects

These projects highlight my expertise in using Machine Learning techniques to solve real-world problems by focusing on computer vision and 3D deep learning models.

AI Fitness Trainer

Introducing an AI fitness trainer that uses computer vision to analyze exercise performance. By capturing videos and assessing each frame, it accurately identifies correct and incorrect movements. With numerical scores for exercise accuracy, it revolutionizes fitness training, helping users track progress, improve form, and achieve their goals effectively.

PoseBias

This project delves into face analysis, using synthetic training data to showcase the significance of camera position. Optimal poses are task-specific, with near-frontal angles proving superior. By comparing pose-specific training to uniform distributions, we reveal the impact of task difficulty on accuracy. Camera positioning is a non-trivial factor in face analysis outcomes.

Finding Best Position in the wild

Unveiling the quest for an optimal camera position in facial image analysis, this project trains cutting-edge models on Real faces (facescape) for various analysis tasks. Through rigorous measurement and evaluation, we aim to uncover the ideal position that maximizes accuracy and performance.
